[SIZE=6][B]Resource Manager Stats Widget for XenForo 2.1.10.2+[/B][/SIZE]

[SIZE=5][B]Description[/B][/SIZE]
This add-on allows showing resource manager statistics in a widget.

[SIZE=5][B]Requirements[/B][/SIZE]
[LIST]
[*]PHP 7.3+
[*]XenForo Resource Manager 2.1.0+
[/LIST]

[SIZE=5][B]Widget Definitions[/B][/SIZE]
[TABLE]
[TR]
[TH]Definition[/TH]
[TH]Description[/TH]
[/TR]
[TR]
[TD]Resource manager statistics ([ICODE]tck_xfrm_stats[/ICODE])[/TD]
[TD]Displays a block which that shows the forum's current resource manager statistics on things like total resources, updates, ratings, files, disk usage and downloads.[/TD]
[/TR]
[/TABLE]

[/TR]
[/TABLE]
